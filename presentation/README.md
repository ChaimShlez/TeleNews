# TeleNews Presentation

מצגת מקצועית ומתומצתת על פרויקט TeleNews.

## 📄 תוכן המצגת

1. **כותרת** - הצגת הפרויקט
2. **סקירה כללית** - מטרות ויכולות מרכזיות
3. **ארכיטקטורה** - מבנה המערכת וזרימת נתונים
4. **מיקרו-סרביסים** - פירוט כל השירותים
5. **טכנולוגיות** - מחסנית הטכנולוגיות
6. **תכונות מתקדמות** - יכולות ייחודיות
7. **פריסה ותפעול** - הוראות הפעלה וניטור
8. **סיכום ועתיד** - הישגים ותוכניות

## 🖨️ המרה ל-PDF

### אופציה 1: דפדפן
1. פתח את הקובץ `telenews-presentation.html` בדפדפן
2. לחץ Ctrl+P (או Cmd+P ב-Mac)
3. בחר "Save as PDF"
4. הגדר:
   - גודל נייר: A4
   - שוליים: רגיל
   - כלול גרפיקה ברקע: כן

### אופציה 2: כלי שורת פקודה
```bash
# עם Puppeteer
npx puppeteer-pdf telenews-presentation.html telenews-presentation.pdf

# עם wkhtmltopdf
wkhtmltopdf telenews-presentation.html telenews-presentation.pdf
```

### אופציה 3: כלים אונליין
- [HTML to PDF Converter](https://www.ilovepdf.com/html-to-pdf)
- [PDF24](https://tools.pdf24.org/en/html-to-pdf)
